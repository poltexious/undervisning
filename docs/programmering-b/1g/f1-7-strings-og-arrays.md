# F1.7 – Strings og arrays

!!! info "Om forløbet"

    - **Timer:** 5 (F-semester)
    - **Studie+-forløb:** SP3 — Toolbox: Genbrugelige byggeklodser
    - **Bog:** *The C# Player's Guide*, Level 12 (arrays). Strings: Level 8 ("Console 2.0")

!!! abstract "Læringsmål"

    Efter forløbet kan du:

    - manipulere tekst i C# og bygge formaterede output-strenge
    - oprette et array, tilgå elementer ved indeks og gennemløbe det med både `for` og `foreach`
