# F1.8 – Stilladsering før Meyer: terninge-mekanik

!!! info "Om forløbet"

    - **Timer:** 2 (F-semester)
    - **Studie+-forløb:** SP3 — Toolbox: Genbrugelige byggeklodser
    - **Bog:** *The C# Player's Guide*, Level 32 (uddrag om `Random`)

!!! abstract "Læringsmål"

    Efter forløbet kan du:

    - bruge `Random`-klassen til at simulere terningeslag
    - kombinere variabler, betingelser og løkker (F1.2, F1.4, F1.5) i ét sammenhængende program

## Om forløbet

*Skriv motivation: dette forløb er bevidst en fælles, ikke-differentieret stilladserings-økt, der samler F1.2/F1.4/F1.5 i ét worked example, som direkte forbereder Meyer-projektet i F1.9. Introducér `Random`-klassen som en "forsmag" på .NET-bibliotekets API'er (underemne 3.1, som først læres i dybden i 2.g).*

## Lektionsplan

*Skriv time-for-time-plan for forløbets 2 timer — opbygget som ét fælles, stepwise worked example (rul-en-terning-program), ikke som individuelt differentierede øvelser.*

## Teori

*Skriv teoriafsnit med C#-kodeeksempler for:*

- *`Random`-klassen: oprettelse og `Next(min, max)`*
- *stepwise opbygning af et rul-en-terning-program: én terning → flere terninger → sammenlign resultater*

## Øvelser

*Dette forløb er ikke differentieret — hele klassen bygger det samme worked example sammen, trin for trin, som direkte stilladsering til Meyer-projektet (F1.9).*

## Leverance
