# F3.5 – Valgfrit/stretch: pathfinding, eget API, AI-værktøjer

!!! info "Om forløbet"

    - **Timer:** 6 (E-semester)
    - **Studie+-forløb:** SP10 — Architecture Mode: Abstraktion og softwarearkitektur
    - **Bog:** *The C# Player's Guide*, Level 30, 33, 34, 48

!!! abstract "Læringsmål"

    Efter forløbet kan du:

    - forklare, hvordan en simpel pathfinding-algoritme fungerer
    - (hvis du vælger det) implementere breadth-first search på et grid
    - designe en lille, genbrugelig klassepakke og dokumentere dens funktionalitet
    - reflektere over, hvornår generativ AI er et gavnligt fagligt værktøj, og hvornår det underminerer din egen læring

*Skriv teoriafsnit (niveaudelt efter spor) om:*
